# Assembly Line—4

An assembly line is a manufacturing process in which
parts are added to a product in a sequential manner to
create a finished product.  Henry Ford popularized the
moving assembly line in 1913, dramatically reducing the
time to build a Model T from over 12 hours to about 93
minutes.

The Toyota Production System transformed the assembly
line concept by introducing flexibility: mixed-model
production (heijunka), the ability to stop the line for
quality issues (jidoka), and pull-based material flow
(kanban).  Where Ford's line was rigid and optimized for
a single product, Toyota's line is adaptable and
optimized for quality and variety.
