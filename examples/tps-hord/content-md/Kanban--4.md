# Kanban—4

Kanban (看板, literally "signboard" or "billboard") is a
scheduling system for lean manufacturing and just-in-time
manufacturing.  Developed by Taiichi Ohno at Toyota, the
system uses visual signals — originally cards attached to
bins of parts — to control the flow of production.

A kanban card signals that a downstream process has
consumed materials and needs replenishment from the
upstream process.  This creates a "pull" system where
production is driven by actual consumption rather than
forecasts.

The system has been widely adopted outside manufacturing,
particularly in software development (Kanban boards),
where it provides visual workflow management using columns
representing process stages.
